# tmov_light_rail

event type: `com.mbta.ocs.tmov_light_rail.v1`

Parsed from OCS TMOV events for light rail trains.

The contents of some fields, such as `speed` and `heading`, rely on the communication method (GPS, AVI, or NVL) by which OCS obtained information about this train, as indicated in the `dataSource` field.
